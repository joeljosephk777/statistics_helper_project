# Statistics Helper Project

A Python-based statistics helper that calculates descriptive statistics and generates visualizations.

## Features
- Mean, median, mode
- Range, variance, standard deviation (sample)
- Quartiles (Q1, Q2, Q3)
- Standard deviation intervals (±1σ, ±2σ, ±3σ)
- Histogram, box plot, and normal distribution overlay (matplotlib)

## Requirements
- Python 3
- matplotlib

## How to Run
```bash
python Statistics.py
